# 🖥️ To-Do List GUI App (Python)

A simple and user-friendly To-Do List application built using Python and Tkinter. This application allows users to manage their daily tasks with a graphical interface instead of a command-line interface.

---

## 🚀 Features

* 🖥️ Graphical User Interface using Tkinter
* ➕ Add new tasks
* 🗑️ Delete selected tasks
* 💾 Persistent storage using file handling (`tasks.txt`)
* 📋 Displays all tasks in a list format
* ⚠️ Error handling for empty input and no selection

---

## 🛠️ Tech Stack

* Python
* Tkinter (built-in GUI library)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/shiva-vashisht/todo-gui-python.git
```

2. Navigate to the project folder:

```bash
cd todo-gui
```

3. Run the application:

```bash
python todo_gui.py
```
