# 🧮 Advanced Calculator

A command-line calculator built using Python that performs multiple mathematical operations with a user-friendly interface.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with zero handling)
- 🔢 Power calculation
- 🧮 Modulus operation
- 🔁 Continuous calculation using previous result
- ❌ Error handling for invalid inputs

---

## 🛠️ Tech Stack

- Python
- VS Code
